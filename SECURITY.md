# Security Policy

## Reporting a vulnerability
Please **do not** open a public GitHub issue for security problems.
Email: arogorn993@gmail.com  
Or use GitHub **Private vulnerability reporting** (if enabled below).
We will acknowledge within a few days and work on a fix before public disclosure when possible.

## Scope
HeatLens desktop app and its GitHub repository. Third-party tools (LibreHardwareMonitor, etc.) are out of scope.
